## Required Equipment

### Indoor Growing
- Metal shelving racks (~$120 per rack, 5 levels)
- LED grow lights (~$40 per shelf level)
- Climate control systems
- Irrigation systems
- Total Investment: $2,000-$5,000

### Outdoor Growing
- Drip irrigation systems
- Raised garden beds
- Basic farming tools
- Storage solutions
- Total Investment: $3,000-$8,000
